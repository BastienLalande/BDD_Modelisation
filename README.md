# Projet Données et Modélisation
## Contexte
Ce projet s’inscrit dans le cadre d’un partenariat avec le ministère de la Transition Écologique pour la création d’un outil permettant de centraliser et interroger des données publiques sur la qualité de l’air dans les grandes villes françaises. L’entreprise ClearData est chargée de développer une base de données relationnelle fonctionnelle comme preuve de concept.

## Objectifs
- Concevoir et implémenter une base de données relationnelle respectant les bonnes pratiques (non-redondance, intégrité, RGPD).
- Gérer :
  - Les agences locales et leur personnel (chefs, agents techniques et administratifs).
  - Les capteurs et leurs relevés mensuels (concentrations de gaz en ppm).
  - Les rapports d’analyse produits par les agents administratifs.
- Fournir des résumés et requêtes SQL pour répondre aux besoins du ministère (ex. liste des agences, relevés par région, rapports par période, statistiques).
